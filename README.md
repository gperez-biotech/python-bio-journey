# python-bio-journey

## Meta
De biotecnólogo a científico de datos biomédicos
## Objetivo
Aprender Python para poder aplicarlo en la biotecnología y a su vez en diferentes ramas específicas como, la bioinformática y sobre todo, en la biotecnología biomédica para lograr medicina de precisión utilizando principalmente Machine Learning.
## Sobre mí
Estudiante de biotecnología (Ecuador) y mis campos de especialización son:
- Medicina de precisión y farmacogenómica.
- Oncología computacional.
- Machine Learning aplicado a la biotecnología biomédica.
- Biotecnología regenerativa.
## Sobre el repositorio
En este repositorio se va a guardar el progreso para lograr la meta mediante el planteamiento de diferentes proyectos para mejorar, tanto mi portafolio, como mi experiencia en la resolución de problemas de manera práctica.
